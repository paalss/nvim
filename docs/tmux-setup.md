Install tmux

```bash
apt install tmux
```

Start a new session
```bash
tmux new
```

Stop session

`[Ctrl]+b d` / `exit`


## Create new panes

To the right

`[Ctrl]+b %`

`[Ctrl]+b "`

## Navigate between panes

`[Ctrl]+b` -- release ctrl -- `[piltast]`

Resize panes

`[Ctrl]+b` -- hold ctrl -- `[piltast]`
